Collection of CGI and Docker CGI Programs

Overview

This repository contains a collection of simple CGI files. The two main purposes of this repositiory are:
  • To provide a set of CGI programs to illustrate the CGI protocol.
    Because of their simplicity, they are easier to understand. These programs can also be used to validate a user's enviornment. It is expect that this programs will be installed on a simple webserver that supports generic CGI programs.

  • To provide a corresponding set of Docker CGI configuration files.
    These configuration files drive the execution of the defined CGI program, via the docker-cgi handler. Refer to the docker-cgi project for more information.

Directory of CGI and Docker CGI Files

CGI FileConfiguration File Description
noop.cgi noop.docker-cgi The most trivial of all CGI programs, it does nothing.
echo-request.cgi echo-request.docker-cgi Emits the components of the query string and the request body.
emit-cgi-env.cgi emit-cgi-env.docker-cgi Emits the environment variables defined by the CGI protocol.
emit-env.cgi emit-env.docker-cgi Emits the environment variables provided by the Web-server, including the CGI variables.
here.cgi here.docker-cgi A CGI program that emits simple HTML document contained within a here document.
cat.cgi cat.docker-cgi Prints out a file based upon the query string, e.g., "cat.cgi?cat.cgi"

Support Files:

  • dockerfile: provides the specification to build the docker image that contains the above files.
  • htaccess.handler: was used to create the .htaccess file for this web folder, and defines which defines the Apache Handler.
  • htaccess.final: is the proposed format of the .htaccess file when the docker-cgi is used under the proposed ContainerCGI module.
